# Maintenance Notes

This directory holds public summaries distilled from earlier local planning records. The root-level agent working files (`task_plan.md`, `findings.md`, and `progress.md`) are local working memory and should not be committed.

Use these notes for stable project context:

- `history-summary.md` summarizes the major maintainability, security, packaging, and QA milestones.
- `roadmap.md` records non-blocking follow-up areas that should guide future maintenance releases.
